# エフェクト処理解説

このドキュメントは、XY_FX_Pad が実際に音声へ行う処理と、聴こえ方を対応させたものです。すべてのFXはパッド押下中に使われ、X / Y / Pressure の3系統を直列に処理できます。複数を重ねるほど派手になる一方、音色変化も大きくなります。

## 操作と音量の安全処理

- **X / Y**: 選択した各FXの主パラメータ。周波数、LFO速度、ディレイ拍などを動かす。
- **Pressure**: `5..120g` を `0..1` に正規化した筆圧。多くのFXではウェット量や深さ、Peak EQ / Ladderでは増幅量や共振を動かす。
- **開始・終了**: パッドのtouch状態を35msの線形クロスフェードに変換する。原音とFX処理済み音をサンプル単位で混ぜるため、押し始め・離し際の急な音量変化を抑える。
- **パラメータ変化**: X / Y / Pressure自体も25msで補間する。センサーの80Hz更新による段差音を減らす。
- **最終保護**: 振幅が0.95を超えた部分だけをソフトサチュレーションする。通常音量は変えず、EQや複数FXで生じる大きなピークを丸める。

## 時間・リズム系

| FX | 音声処理 | 聴こえ方 / 主な操作 |
| --- | --- | --- |
| Gate | BPMの16分周期で音量を台形状に開閉する。Xは開いている長さと深さ、Pressureは原音との混合量。 | 断続的な刻み。SDVX風のFXロング、点・止めの強調に向く。 |
| Echo | BPM同期の円形ディレイ。Xで音符長、Pressureでフィードバックとウェット量。 | 拍に合った反復。強い筆圧ほど残響が増える。 |
| Reverb | JUCEのリバーブ。Xでroom sizeとdamping、Pressureでwet量。 | 空間が広がる。払い・余韻を大きく見せる。 |
| Tremolo | 正弦LFOで音量を周期変調。Xで速度と深さ。 | 音が脈打つ。速い値ではバタついたリズム感になる。 |
| Roll | 直前の音をBPM同期の短い区間でループ再生。Xで `1/8..4 beats` を選ぶ。 | Retrigger / stutter。Pressure枠に置くと、筆圧でループ長を選ぶ。 |

## フィルター・EQ系

| FX | 音声処理 | 聴こえ方 / 主な操作 |
| --- | --- | --- |
| Filter | Xの左側は共振付きLPF、中央はほぼ素通し、右側は共振付きHPF。 | DJフィルター。横方向の筆運びに最も分かりやすく対応する。 |
| Low Pass | 指定周波数より高い成分を減らすIIRフィルター。Xで180Hzから19kHzまで動かす。 | 左側ほど暗く・こもる。 |
| High Pass | 指定周波数より低い成分を減らすIIRフィルター。 | 値を上げるほど低音が抜け、細く明るい質感になる。 |
| Band Pass | 中心帯域だけを通すIIRフィルター。音量低下を補うため2.2倍のゲイン補正を行う。 | 電話・ワウ風。極端な値ではピークに注意。 |
| Notch | 中心帯域だけを削るIIRフィルター。 | 特定の帯域に穴が開く、薄く揺らいだ音。 |
| Peak EQ | 指定周波数をベル型にブースト。Xが120Hzから8kHz、Pressureが約+0.5から+14dBとQを動かす。 | 筆圧で音の芯を前へ出せる。`Ink Accent`の中心。 |
| Low Shelf | 250Hzより低い全体を-12から+12dB。Xがカット／ブーストを選ぶ。 | 太さ、キック、ベースの量を調整する。 |
| High Shelf | 3.5kHzより高い全体を-12から+12dB。 | 明るさ、空気感、シャリつきを調整する。 |
| Ladder | 24dB/octのラダーローパス。Xがcutoff、Pressureがresonanceとdrive。 | 押すほど「ワウ」「うねり」が強くなる。Wobble表現向け。 |

## 変調・質感系

| FX | 音声処理 | 聴こえ方 / 主な操作 |
| --- | --- | --- |
| Flanger | 数msの遅延をLFOで揺らし、遅延音を原音へ戻す。Xが速度、Pressureが深さ・wet。 | ジェット機のような上昇／下降感。 |
| Phaser | オールパス段による位相差をLFO変調。Xが速度・中心帯域、Pressureがfeedback・mix。 | 周波数の穴が移動する、渦巻くような音。 |
| Chorus | 少し長い変調ディレイを混ぜる。Xが速度／遅延、Pressureが深さ／mix。 | 音が太く広がる。 |
| Auto Pan | LFOで左右チャンネルのゲインを逆方向に動かす。Xが速度・振り幅。 | ステレオ空間を左右へ移動する。ヘッドホン向き。 |
| Drive | `tanh`によるソフトクリップ。Xがキャラクター、Pressureがdrive量とwet。 | 太い歪み。高いPressureでは音量を抑えている。 |
| BitCrusher | サンプル振幅を少ない段階へ丸める。Xで量子化段数とwetを動かす。 | デジタルで粗いローファイ感。 |
| Compressor | threshold / ratio / attack / releaseをXで同時制御し、最大+6dBのメイクアップゲインを加える。 | 音量差が小さくなり、密度が上がる。 |

## プリセットの考え方

- **書道系**は、位置と筆圧の因果関係を保つため、原則として1〜2個のFXに絞る。
- **SDVX inspired系**は、X/Yを左右つまみ、PressureをFXボタン長押しとして使う。特に `SDVX Laser Wobble`、`SDVX FX Gate`、`SDVX Retrigger` が操作感を掴みやすい。
- **Air Brush / Noisy Filter**のような3FX直列は演出用。曲のブレイクや短い筆画で使うと音が濁りにくい。
