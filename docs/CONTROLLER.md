# 物理コントローラー仕様

XY_FX_Pad の物理コントローラーは、4点の荷重から2次元座標を推定する圧力式FXパッドです。センサー読み取り、座標推定、GUI描画を分離し、音声処理側には軽量なXY/touch状態だけを渡します。

## ハードウェア

- MCU: ESP32-S3
- 荷重センサー: ロードセル x4
- ADC: HX711 x4
- HX711 RATE: 80Hz（RATEピンをVCC側）
- ロードセル想定: SC616C 500g
- シリアル: 115200 baud
- 物理フィールド比率: 3:2

4つのロードセルはフィールド四隅に対応します。

| CSVフィールド | 位置 |
| --- | --- |
| `raw0` | 右上 |
| `raw1` | 左上 |
| `raw2` | 左下 |
| `raw3` | 右下 |

## ファームウェア

[firmware.ino](../firmware.ino) はESP32-S3上で4つのHX711を読み、約80HzでCSVを出力します。

```text
raw0,raw1,raw2,raw3
```

- 出力単位: HX711の生の符号付き24-bit値
- 出力形式: CSVのみ

## ControllerBridge

`ControllerBridge` はシリアル入力を読み、4点荷重をXY/touchへ変換してUDP送信します。

```sh
build/ControllerBridge
```

デフォルト設定:

- シリアルポート: `/dev/cu.usbmodem1101`
- ボーレート: `115200`
- UDP送信先: `127.0.0.1:45454`
- touch閾値: `5g`

接続ごとに、パッドを無荷重にした状態で次の処理を行います。

- 最初の80サンプル（約1秒）をウォームアップとして破棄
- 続く140サンプル（約2秒）のチャンネル別メジアンをゼロ点に採用
- SC616C 500g / 0.7mV/V、HX711 gain 128の定格からgへ換算
- チャンネル `-50..250g`、合計 `-20..250g` の物理範囲外を破棄
- チャンネル `20g` / 合計 `30g` 以上の急変は、近い値が連続2サンプル届くまで保留し、単発スパイクをタッチとして送らない
- 確認済みの急変だけをチャンネル `40g` / 合計 `60g` までにクランプして追従
- 採用値に3点メジアンとEMA（`alpha=0.35`）を適用
- シリアル無通信が約3秒続くと再接続

実行中のショートカット:

- `c` … ゼロ点を再キャリブレーション（パッドを空にしてから押す）
- `q` … 終了

ロードセルの定格またはHX711ゲインが異なる場合は、`Tools/ControllerBridge.cpp` 冒頭の定数を実機の値へ変更します。

コンソールには座標または未検出状態を表示します。

```text
XY 0.523,0.417 total=34.2g
検出なし total=2.1g
```

## 座標推定

フィルター後の合計荷重から重心を求めます。

```text
total = g0 + g1 + g2 + g3
x = (topRight + bottomRight) / total
y = (topLeft + topRight) / total
```

`total < 5g` の場合はtouchなしとして扱います。

実機の外周には有効操作できないマージンがあるため、Bridge側で5%の逆マージン補正を行います。物理的に届く `0.05..0.95` 付近の範囲を、ソフトウェア上の `0.0..1.0` に展開します。補正を控えめにして、端へ急に吸い付くような位置変化を抑えています。

## アプリ連携

JUCEアプリ本体はシリアルポートを開きません。`ControllerReceiver` がUDPで以下の形式を受信します。

```text
x,y,touch,total
```

音声処理側の `padX`, `padY`, `padPressure`, `effectsActive` は受信サンプルごとに更新します。
`totalGrams` は約 `5..120g` を筆圧 `0..1` に正規化し、Peak EQ のゲインや Ladder のレゾナンス、その他FXのウェット量に使います。
GUI描画は20Hz timerで最新サンプルだけを反映し、80Hzの制御応答と軽い描画負荷を両立します。
UDP受信が200ms途切れた場合は、アプリ側が自動的にtouchを解除するため、USB抜けやBridge停止時にFXが掛かり続けません。
